## 描述

HTTP `504 Gateway Timeout` 服务器错误响应码表示服务器在充当网关或代理时，未能及时从上游服务器获取完成请求所需的响应。

<aside class="info"><strong>注意：</strong> 网关在网络中可能指代不同的事物，504 错误通常不是您可以修复的问题，而是需要 Web 服务器或您尝试访问的代理进行修复。</aside>
