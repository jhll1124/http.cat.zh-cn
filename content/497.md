## 描述

<aside class="warning"><strong>非官方：</strong> 此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP 响应状态码 `497 HTTP Request Sent to HTTPS Port` 是一个非官方的 HTTP 状态码，特定于 nginx，由服务器返回以指示常规 HTTP 请求已发送到 HTTPS 端口。

当收到 `497 HTTP Request Sent to HTTPS Port` 状态码时，HTTP 请求可能是有效的，但服务器不愿意处理它，因为它是发送到 HTTPS 端口的。这与错误 `400 Bad Request` 相关。
