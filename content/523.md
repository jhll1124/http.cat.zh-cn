## 描述

<aside class="warning"><strong>非官方：</strong> 此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP 响应状态码 `523 Origin Is Unreachable` 是一个非官方的 HTTP 状态码，特定于 [Cloudflare](https://cloudflare.com)。

当 Cloudflare 无法联系您的源 Web 服务器时，会发生 `523` 错误。这通常发生在 Cloudflare 和源 Web 服务器之间的网络设备没有到源 IP 地址的路由时。
