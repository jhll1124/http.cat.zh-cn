## 描述

HTTP `431 Request Header Fields Too Large` 响应状态码表示服务器拒绝处理请求，因为请求的 HTTP 标头太长。在减小请求标头的大小后，可以重新提交请求。

当请求标头的总大小太大，或者单个标头字段太大时，可以使用 `431`。为了帮助遇到此错误的人，请在响应正文中指出两者中的哪一个问题是问题所在 —— 理想情况下，还包括哪些标头太大。这允许用户尝试修复问题，例如通过清除他们的 cookie。

如果出现以下情况，服务器通常会返回此状态：

- `Referer` URL 太长
- 请求中发送了太多 Cookie
