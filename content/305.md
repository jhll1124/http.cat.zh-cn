## 描述

<aside class="alert"><strong>已弃用：</strong> 由于安全考虑，HTTP 305 Use Proxy 状态码已被弃用，许多 HTTP 客户端会忽略此状态码。</aside>

HTTP 响应状态码 `305 Use Proxy` 是一个已弃用的 HTTP 状态码，由源服务器返回，指示请求的资源只能通过代理服务器访问。

`305 Use Proxy` 状态码用于通知客户端必须通过代理服务器获取请求的资源，并在 `Set-Proxy` HTTP 标头或 `Location` HTTP 标头中包含该代理地址。
