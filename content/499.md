## 描述

HTTP 响应状态码 `499` 是一个非官方的 HTTP 状态码，特定于 ArcGIS 的 `499 Token Required` 和 nginx 的 `499 Client Closed Request`。

#### ArcGIS

当收到 `499 Token Required` 状态码时，服务器期望 HTTP 请求中包含令牌，但未提交。

#### nginx

当收到 `499 Client Closed Request` 状态码时，这意味着客户端在服务器完成任务之前提前关闭了 HTTP 连接。因此，无法发送最终响应，并且此状态码仅在 nginx 日志中可用。
