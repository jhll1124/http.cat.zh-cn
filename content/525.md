## 描述

<aside class="warning"><strong>非官方：</strong> 此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP 响应状态码 `525 SSL Handshake Failed` 是一个非官方的 HTTP 状态码，特定于 [Cloudflare](https://cloudflare.com)。

`525` 错误指示 Cloudflare 和源 Web 服务器之间的 SSL 握手失败。当满足这两个条件时，会出现错误 `525`：

- Cloudflare 和源 Web 服务器之间的 SSL 握手失败，并且
- 在您的 Cloudflare SSL/TLS 应程序的概述选项卡中设置了 Full 或 Full (Strict) SSL。

<aside class="info"><strong>注意：</strong> 如果您的托管提供商经常更改源 Web 服务器的 IP 地址，请参阅 Cloudflare 关于动态 DNS 更新的文档。</aside>
