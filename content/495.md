## 描述

<aside class="warning"><strong>非官方：</strong> 此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP `495 SSL Certificate Error` 是一个非官方的客户端错误，特定于 nginx，由服务器返回以指示客户端证书验证存在错误。
