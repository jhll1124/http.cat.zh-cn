## 描述

<aside class="warning"><strong>非官方：</strong> 此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP 响应状态码 `522` 是一个非官方的 HTTP 状态码，特定于 [Cloudflare](https://cloudflare.com)。

当 Cloudflare 联系源 Web 服务器超时时，会发生 `522` 错误。根据它们发生在 Cloudflare 和源 Web 服务器之间的时间，两个不同的超时会导致 HTTP 错误 `522`：

- 在建立连接之前，源 Web 服务器在 Cloudflare 发送 `SYN` 后 15 秒内没有向 Cloudflare 返回 `SYN+ACK`。
- 建立连接后，源 Web 服务器在 90 秒内未确认（`ACK`）Cloudflare 的资源请求。
