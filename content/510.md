## 描述

HTTP `510 Not Extended`
响应状态码在 HTTP 扩展框架的上下文中发送，该框架在 RFC 2774 中定义。

在该规范中，客户端可以发送包含扩展声明的请求，该声明描述要使用的扩展。如果服务器接收到此类请求，但不支持该请求的任何描述的扩展，则服务器以 `510` 状态码响应。
