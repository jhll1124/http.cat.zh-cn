## 描述

HTTP `207 Multi-Status` 响应码表示可能存在混合响应。

响应正文是具有 `multistatus` 根元素的 `text/xml` 或 `application/xml` HTTP 实体。XML 正文将列出所有单独的响应代码。


  <aside class="info"><strong>注意：</strong> 返回资源集合的能力是 WebDAV 协议的一部分（访问 WebDAV 服务器的 Web 应用程序可能会接收到它）。
  访问网页的浏览器永远不会遇到此状态码。</aside>
