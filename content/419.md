## 描述

<aside class="warning"><strong>非官方：</strong> 此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP 响应状态码 `419 Page Expired` 是一个非官方的 HTTP 状态码，主要由 Laravel PHP 框架用于指示 CSRF 令牌不匹配。
