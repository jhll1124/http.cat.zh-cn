## 描述

HTTP `226 IM Used` 响应状态码，在增量编码的上下文中，是服务器设置的状态码，用于指示它正在返回对其接收到的 GET 请求的增量。IM 代表 *Instance Manipulations*（实例操作），这是用于描述生成增量的算法的术语。

使用增量编码时，服务器响应 `GET` 请求时使用相对于给定基本文档（而不是当前文档）的差异（称为增量）。客户端使用 `A-IM:` HTTP 标头指示要使用的差分算法，并使用 `If-None-Match:` 标头向服务器提示其获得的最后一个版本。服务器生成增量，通过具有 `226` 状态码的 HTTP 响应将其发回，并包含 `IM:`（带有使用的算法名称）和 `Delta-Base:`（带有与增量关联的基本文档相匹配的 `ETag`）HTTP 标头。

<aside class="info"><strong>注意：</strong> 浏览器不支持使用 HTTP 进行增量编码。此状态码由特定客户端使用的自定义服务器发回。</aside>
