## 描述

<aside class="warning"><strong>非官方：</strong> 此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP 响应状态码 `530 Site Frozen` 是一个非官方的 HTTP 状态码，特定于 Cloudflare 和 Pantheon。

#### Cloudflare

当用户和网站服务器之间出现问题时，Cloudflare 使用 `530` 状态码。错误取决于具体情况，并附带更详细的 HTTP 状态码 `1XXX`。

#### Pantheon

当非活动的 Sandbox Site 拒绝 HTTP 请求时，Pantheon 返回 `530 Site Frozen` 状态码。可以通过在 Pantheon 用户控制面板中重新激活站点来解决此问题。
