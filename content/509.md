## 描述

<aside class="warning"><strong>非官方：</strong> 此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP 响应状态码 `509 Bandwidth Limit Exceeded` 是一个非官方的 HTTP 状态码，特定于 Apache Web Server 和 cPanel，指示 Web 托管客户端已在服务器端超出带宽分配。

当收到 `509 Bandwidth Limit Exceeded` 状态码时，首先意味着 Web 托管客户端正在使用带宽有限的连接或托管方法。
