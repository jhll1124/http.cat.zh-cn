## 描述

<aside class="warning"><strong>非官方：</strong> 此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP 响应状态码 `450 Blocked by Windows Parental Controls` 是一个非官方的 HTTP 状态码，特定于 Microsoft，由服务器返回以指示家长控制已打开并阻止对请求资源的访问。

当收到 `450 Blocked by Windows Parental Controls` 状态码时，意味着 Windows 家长控制已启用并拒绝访问目标资源。
