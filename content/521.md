## 描述

<aside class="warning"><strong>非官方：</strong> 此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP 响应状态码 `521 Web Server Is Down` 是一个非官方的 HTTP 状态码，特定于 [Cloudflare](https://cloudflare.com)。

当源 Web 服务器拒绝来自 Cloudflare 的连接时，会发生此错误。

### 另请参阅
