## 描述

<aside class="alert"><strong>已弃用：</strong> HTTP 214 Transformation Applied 状态码和 Warning HTTP 标头已被弃用。</aside>

HTTP `214 Transformation Applied` 是一个非官方状态码，与 [Warning HTTP 标头](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Warning) 结合使用，并包含在 HTTP 响应消息中，以指示已对表示形式应用了转换。

如果代理对表示形式应用任何转换（例如更改 `content-coding`、`media-type` 或修改表示形式数据），则必须添加此状态码和 `Warning` 标头，除非响应中已存在 `Warning` 标头。
