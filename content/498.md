## 描述

<aside class="warning"><strong>非官方：</strong> 此 HTTP 状态码未由任何 RFC 指定。</aside>

HTTP 响应 `498 Invalid Token` 是一个非官方的 HTTP 状态码，特定于 ArcGIS，指示 HTTP 请求包含令牌但未被接受。

当收到 `498 Invalid Token` 状态码时，客户端在 HTTP 请求中包含了所需的令牌，但由于令牌被认为无效，服务器不接受它。
